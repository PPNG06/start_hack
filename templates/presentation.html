About us



What is Switch Learning ?

We authenticate and simplify the administrative processes for lifelong learners. Through Blockchain technology we provide a service that allows academic documents be both instantly verifiable, and substantially less expensive to store and transfer. We aim to develop new ways for people, institutions and companies to exchange and store academic credentials. By using distributed ledger technology we can finally access a safer and more streamlined approach to data storage, resulting in security and transparency being built into the core of the solution.


Why this project ?

We believe that educational records are not only important but should also be handled with care. We guide individuals in every step of their formal education and beyond.


How does it work?

During the process, the file itself also gets stored on IPFS, a peer-to-peer, decentralized and distributed system for storing and accessing files. On the other hand, the metadata id of the file is stored on the Ethereum blockchain. By providing an exchange platform, moving, and encrypting the data into IPFS and Ethereum, we are offering a verification tool for each file to allow every useful educational record to be gathered onto one platform.
